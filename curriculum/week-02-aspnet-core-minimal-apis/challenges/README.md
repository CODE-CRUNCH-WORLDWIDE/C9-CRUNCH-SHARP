# Week 2 — Challenges

The exercises drill basics. **Challenges stretch you.** This week's challenge takes ~2 hours and produces something you can commit to your portfolio: a five-endpoint typed Todo API with validation, OpenAPI, and DI-driven persistence.

## Index

1. **[Challenge 1 — The Five-Endpoint Todo API](challenge-01-the-five-endpoint-todo-api.md)** — build a complete typed CRUD API for a Todo domain using `MapGroup`, `TypedResults`, data-annotation validation, and a DI-registered store. (~120 min)

Challenges are optional. If you skip them, you can still pass the week. If you do them, you'll be measurably ahead — and the five-endpoint pattern here reappears in this week's mini-project (the Ledger REST service) and in Week 5's Sharp Notes API.
