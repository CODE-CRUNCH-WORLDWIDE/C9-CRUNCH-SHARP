# Week 1 — Challenges

The exercises drill basics. **Challenges stretch you.** Each one takes 60–120 minutes and produces something you can commit to your portfolio.

## Index

1. **[Challenge 1 — Fluent builder with records and `with`](challenge-01-fluent-builder.md)** — build a small, type-safe fluent builder API using records and `with`-expressions. (~90 min)

Challenges are optional. If you skip them, you can still pass the week. If you do them, you'll be measurably ahead of someone who didn't — and the builder pattern here reappears in Week 5 (Minimal API options) and Week 6 (EF Core fluent configuration).
